## COSMO

COSMO is a moderation bot with easy configuration along with swift command execution.

## Acquiring a licence

1. Buy the COSMO licence from the product hub,
2. Create a ticket in are support channel with proof of purchase,
3. After we have confirmed the purchase you will be whitelisted and given and URL to invite the bot.

## Guild Setup

1. After the bot has joined run the `/settings` command to configure further settings,
2. When you run the command find `Set Mod Role` which allows you to set a guild role as a mod role which lets users with that role `kick, ban, warn etc`,
3. You will also see `Set Log Channel` and make that a private channel so you can see actions by mods.
