### Information

COSMO is a moderation bot with easy configuration along with swift command execution.

### Acquiring a licence

1. Go to the [Product Hub](https://www.roblox.com/games/134688955551796/Product-Hub) and buy `COSMO` .
2. Create a ticket in the [Support Channel](https://discord.com/channels/1439762088187924502/1439791029589905613) with proof of purchase
3. Then wait for staff to reply and verify you have bought it.
4. After you are verified you will get a invite URL so the bot can be added to the server of choice.

### Server setup

1. When the bot has joined run `/settings` .
2. Find the button called `Set Mod Role` so you can set a guild role for moderators which allows access to `kick, ban, warn etc` .
3. After find `Set Log Channel`  button and set the channel to a private channel so you can monitor moderator actions.
