# COSMO Information

### Information

1. COSMO is a moderation bot with easy configuration along with swift command execution.
2. Price: `R$ 99`

### CORE

1. Highly customizable moderation modules.
2. Efficient command handling with minimal latency.
3. Supports automated actions such as muting, warning, and logging.
4. Modular structure allowing quick updates and feature expansion.
5. Designed to integrate smoothly with existing server systems.

### Features

1. Advanced Moderation: Filters with live scanning.
2. Command System: Fast responses with intuitive command syntax.
3. Logging Suite: Tracks infractions, joins/leaves, message edits, and deletions.
4. Role Management: Automated permission and role assignments.
5. Security Tools: Anti-raid protection, spam detection, and link filtering.

# Setup

### Acquiring a licence

1. Go to the [Product Hub](https://www.roblox.com/games/134688955551796/Product-Hub) and buy `COSMO` .
2. Create a ticket in the [Support Channel](https://discord.com/channels/1439762088187924502/1439791029589905613) with proof of purchase
3. Then wait for staff to reply and verify you have bought it.
4. After you are verified you will get a invite URL so the bot can be added to the server of choice.

<Danger>
  WARNING : TRYING TO GET A LICENCE WITHOUT PAYING WILL HAVE YOU BLACKLISTED.
</Danger>

### Server setup

1. When the bot has joined run `/settings` .
2. Find the button called `Set Mod Role` so you can set a guild role for moderators which allows access to `kick, ban, warn etc` .
3. After find `Set Log Channel`  button and set the channel to a private channel so you can monitor moderator actions.
