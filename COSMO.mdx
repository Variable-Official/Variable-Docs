# COSMO Information

### Information

1. COSMO is a moderation bot with easy configuration along with swift command execution.
2. Price: `R$ 99` 

### CORE

1. Highly customizable moderation modules.
2. Efficient command handling with minimal latency.
3. Supports automated actions such as muting, warning, and logging.
4. Modular structure allowing quick updates and feature expansion.
5. Designed to integrate smoothly with existing server systems.

### Features

1. Advanced Moderation: Filters with live scanning.
2. Command System: Fast responses with intuitive command syntax.
3. Logging Suite: Tracks infractions, joins/leaves, message edits, and deletions.
4. Role Management: Automated permission and role assignments.
5. Security Tools: Anti-raid protection, spam detection, and link filtering.

<Warning>
  Some features are planned for later updates.
</Warning>
